Example system names = INFERENCE | CORE | STD

MBASE_<SYSTEM_NAME>_SYS_STRING : string ---> Name of the library
MBASE_<SYSTEM_NAME>_PATH_STRING : string ---> Include path of the library
MBASE_<SYSTEM_NAME>_VERSION : string ---> Version string
MBASE_<SYSTEM_NAME>_INCLUDE_DEPENDS : list ---> Include paths (non installed)
MBASE_<SYSTEM_NAME>_LIB_DEPENDS : list ---> Dependant libraries
MBASE_<SYSTEM_NAME>_INCLUDE_INSTALL_PATH : str ---> Installation source for header files
MBASE_<SYSTEM_NAME>_INCLUDE_STABLE_FILES : list ---> List of header files
MBASE_<SYSTEM_NAME>_INCLUDE_INSTALL_FILES : list ---> List of MBASE_<SYSTEM_NAME>_INCLUDE_INSTALL_PATH + header files
